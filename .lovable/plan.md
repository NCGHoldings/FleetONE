

# How to Add a Tax Invoice Template to the Document Template System

## How It Works Today

The **Tax Invoice** for vehicle sales (Yutong, Sinotruck, Light Vehicle) is generated by a **standalone shared generator** (`sri-lanka-tax-invoice-generator.ts`) that produces the Sri Lankan government format directly in code. It is **not** part of the central Document Template system (the one managed in **Settings â†’ Document Templates**).

The central template system manages 12 finance document types (AR Invoice, AP Invoice, Receipts, Vouchers, etc.) stored in the `document_templates` database table. These are customizable via the Template Manager UI and initialized by the Template Initializer button.

## What Needs to Happen

To make the Sri Lankan Tax Invoice available as a **manageable template** in the Document Template system, these changes are needed:

### 1. Add a new `document_template_types` record
Insert a new template type row in the database:
- `type_code`: `tax_invoice`
- `type_name`: `Sri Lanka Tax Invoice`
- `module`: `general` (or a new `vehicle_sales` module)
- `available_placeholders`: JSON array of all supported placeholders (e.g., `{{supplier_tin}}`, `{{purchaser_name}}`, `{{total_amount}}`, etc.)

### 2. Add Tax Invoice to the Template Seeder
In `src/lib/document-template-seeder.ts`:
- Create a new `generateTaxInvoiceTemplate()` function that produces the Sri Lankan government-mandated layout as an HTML template with `{{placeholder}}` tokens
- Register it in `defaultTemplates` and `templateDisplayNames`

### 3. Update the Template Initializer
The existing `TemplateInitializerButton` already loops through all `defaultTemplates` entries and matches them to `document_template_types`. Once steps 1 and 2 are done, clicking **"Initialize All Templates"** (or **"Replace All Templates"**) in Settings will automatically create the tax invoice template for each company.

### 4. Connect Vehicle Sales Generators to the Template System
Update the three invoice generators (`yutong-order-invoice-generator.ts`, `sinotruck-order-invoice-generator.ts`, `lightvehicle-order-invoice-generator.ts`) to:
- Fetch the `tax_invoice` template from `document_templates` when generating a tax invoice
- Use `replacePlaceholders()` to fill in the template with actual data
- Fall back to the current hardcoded Sri Lankan format if no template exists
